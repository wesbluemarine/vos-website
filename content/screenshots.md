## Desktop

VitruvianOS runs the Haiku desktop environment on top of a real-time patched Linux kernel.
Screenshots will be added here as the system reaches a more polished state.

Want to contribute a screenshot? Open a pull request on [GitHub](https://github.com/VitruvianOS/Vitruvian).

## Deskbar

The Deskbar provides the application launcher, running application list, and system tray — styled with the VitruvianOS branding.

## Tracker

The Tracker file manager supports live queries, extended attributes, and spatial window management, faithful to the original BeOS design.
